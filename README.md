# react-flutter-nav

> A trending navigation style in Flutter implemented using ReactJS.

### Status

:rocket: Completed

![Demo Image](/demo.png)

## Installation

- Clone the repo
- `cd` into dir and run `yarn install`
- To start the project, run `yarn start`
- Visit `http://localhost:3000`
